# PRUEBATECNICA-INVENTARIOPRODUCTOS
PRUEBA TECNICA INVENTARIO DE PRODUCTOS CON .NET CORE 8, BLAZOR  Y SQL-SERVER

#REQUERIMIENTOS INCLUIDOS

• La aplicación debe contar con inicio de sesión (usuario y contraseña) previamente 
registrado. 
• El usuario operador tendrá solo la opción de realizar ventas. 
• Desarrollo de una aplicación web que consuma API Rest de productos, esta debe 
realizar mantenimiento (CRUD) 
• La aplicación web debe realizar la búsqueda de productos por código y agregar en 
detalle de venta a realizar, esta debe realizar los cálculos en pantalla. 
• El sistema debe guardar los datos de la venta haciendo uso del API Rest. 
• Creación de tabla productos, ventas y detalleventas, hacer uso de procedimientos 
almacenados. 
• La aplicación debe mostrar notificaciones de la operación realizada y contar con 
manejo de errores. 
• Generar un reporte en PDF y excel de ventas realizadas.
